# Pariox Website

Source for [pariox.co](https://pariox.co) — the homepage of Pariox, a service that builds private, fine-tuned AI models for MSPs and regulated businesses that can't send their data to OpenAI.

## What's in this repo

- `index.html` — the complete single-page site. HTML, CSS, and JavaScript are all inline for simplicity; no build step, no dependencies.
- `CNAME` — custom domain configuration for GitHub Pages.

## About Pariox

Pariox builds custom AI assistants for managed service providers. The model is trained on your own ticket history, internal documentation, and escalation procedures — so it answers in your team's format, knows your systems by name, and runs entirely on your own hardware. No third-party cloud involvement. No subscription dependency.
